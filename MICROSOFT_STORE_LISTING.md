## Description (English)

### Insait Translator Deutsch - Your Smart German Translation & Voice Assistant

Insait Translator Deutsch is a powerful, user-friendly desktop application for anyone working with the German language. Whether you need translations, want to listen to German text, or create MP3 audio files — this app has you covered.

**Important:** This app uses free translation services where available. Free services can have usage limits/quotas that may change over time — please check the providers’ websites for current details. Google Translate is used as a fallback provider and typically supports up to ~5,000 characters per single translation.

#### 🌟 Key Features

**📝 Smart Translation**
- Fast and accurate translation to German
- Multiple translation providers with automatic fallback (MyMemory, Google Translate)
- Support for Google Gemini API for enhanced accuracy
- Auto-detection of source language
- **Note:** Free translation services have usage limitations (check provider websites for details)
- Google Translate (fallback) supports up to ~5,000 characters per single translation

**🔊 German Text-to-Speech & MP3 Creation**
- High-quality German voice synthesis using Piper TTS
- Natural-sounding thorsten voice model
- **Listen to any German text** — no translation needed!
- **Export any German text to MP3** for offline listening, learning, or sharing
- Perfect for pronunciation practice and language learning
- Create audiobooks, study materials, or voice notes in German

**📚 Translation History**
- Automatic saving of all translations
- Quick access to previous translations
- Search through your translation history

**🗂️ Multiple Workspaces**
- Up to 3 simultaneous workspaces
- Switch between different translation projects
- Browser-like tab interface

**🌍 Multilingual Interface**
- Available in 5 languages: Ukrainian, English, German, Russian, Turkish
- Easy language switching

**🎨 Modern Design**
- Beautiful warm color palette
- Clean, intuitive interface
- Windows Fluent Design elements

#### 💼 Perfect For
- Language learners preparing for German exams (A1-B2)
- Content creators needing German audio
- Business professionals
- Translators and interpreters
- Students and teachers
- Anyone who wants to hear German text spoken aloud

#### 🔒 Privacy First
- No account required
- Translations processed through secure APIs
- Local storage for history and settings
- No personal data collection

#### 🖥️ System Requirements
- Windows 10/11
- .NET 10.0 Runtime
- Internet connection for translation (TTS works offline after initial setup)

**📋 Translation Service Limits**
- Free translation services have daily/monthly usage limits
- Google Translate (fallback): ~5,000 characters per single translation
- For unlimited translations, consider using Google Gemini API (requires API key)
- Check individual service websites for current limitation details
