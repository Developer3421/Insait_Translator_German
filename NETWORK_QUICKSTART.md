# 🌐 Швидкий старт: Доступ з мережі

## Для адміністратора (хто запускає додаток):

### Варіант 1: Автоматичне налаштування (рекомендовано)

Клацніть правою кнопкою миші на `Setup-NetworkAccess.ps1` і виберіть **"Запустити з PowerShell як адміністратор"**.

Цей скрипт автоматично:
- Додасть правила брандмауера для портів 4200 і 4201
- Налаштує URL ACL резервації для HTTP.sys
- Покаже вашу IP адресу

### Варіант 2: Ручне налаштування

#### 1. Дозвольте порти в брандмауері Windows

Відкрийте PowerShell **як адміністратор** і виконайте:

```powershell
New-NetFirewallRule -DisplayName "Insait Translator Backend" -Direction Inbound -LocalPort 4200 -Protocol TCP -Action Allow
New-NetFirewallRule -DisplayName "Insait Translator React UI" -Direction Inbound -LocalPort 4201 -Protocol TCP -Action Allow
```

#### 2. Додайте URL ACL резервації

```powershell
netsh http add urlacl url=http://+:4200/ sddl=D:(A;;GX;;;WD)
netsh http add urlacl url=http://+:4201/ sddl=D:(A;;GX;;;WD)
```

### 3. Дізнайтеся вашу IP адресу

```powershell
ipconfig
```

Знайдіть рядок з **IPv4 Address**, наприклад: `192.168.1.100`

### 4. Запустіть додаток

Просто запустіть Insait Translator Deutsch як зазвичай.

### 5. Поділіться адресою з користувачами

Скажіть їм відкрити у браузері:
```
http://[ВАША-IP]:4201
```

Наприклад:
```
http://192.168.1.100:4201
```

## Усунення проблем:

### "The request is not supported" при запуску

Це означає, що HTTP.sys не має дозволу слухати на вказаних портах. 
Запустіть `Setup-NetworkAccess.ps1` як адміністратор або виконайте:

```powershell
netsh http add urlacl url=http://+:4200/ sddl=D:(A;;GX;;;WD)
netsh http add urlacl url=http://+:4201/ sddl=D:(A;;GX;;;WD)
```

### "Не можу підключитися"

1. Переконайтеся, що ви в одній мережі Wi-Fi
2. Перевірте IP адресу сервера (вона може змінитися)
3. Перевірте, чи запущений додаток на сервері
4. Спробуйте вимкнути VPN на обох пристроях

### "ERR_CONNECTION_REFUSED"

Брандмауер блокує підключення. Запустіть `Setup-NetworkAccess.ps1` як адміністратор.

### "Backend unavailable"

Додаток на сервері не запущений або зависло. Перезапустіть його.
